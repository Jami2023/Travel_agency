# Travel_agency

Bangla Voyage
A modern, responsive travel agency website designed to showcase beautiful tourist destinations in Bangladesh.

✨ Features
Hero Section: Engaging introduction to explore Bangladesh.

Search Functionality: Filter tours by destination, type, and region.

Destinations Gallery: Interactive cards displaying popular tourist spots like Sajek Valley, Bandarban, and Sylhet.

Testimonials: Social proof section featuring traveler feedback.

Responsive Design: Optimized for seamless viewing across desktops and mobile devices.
